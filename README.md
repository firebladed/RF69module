# RF69module
Simple PCB for HopeRF RFM69HCW exposing SPI interface and DIO to access SPI Bus

- RFM69HCW SPI interface accessable on Chip Select 0
- RFM69HCW DIO accessable via MCP23S08 IO expander on  Chip Select 1

Pin Connections 
1. \~ALERT\~
2. GND
3. SPI_MISO
4. SPI_MOSI
6. SPI_SCLK
7. SPI_CS0
8. SPI_CS1
9. GND
10. 3V3
